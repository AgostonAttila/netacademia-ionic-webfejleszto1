# Netacademia Ionic tanfolyam

https://netacademia.hu/Tanfolyam/2017ionic-ionic---appra-fel

Webes alapokról indulsz és szeretnél mobil appot készíteni? Nem akadály. Az Ionic az egyik legismertebb rendszer, amellyel az ismert alapokról tudsz elrugaszkodni. Ráadásul lehet a cél Android vagy IOS, esetleg WP alapú eszköz is.


## Telepítés és beállítás
- Ismerekedés az Ionic CLI-vel.
- Egy tipikus Ionic alkalmazás felépítése és kezelése.

## Ionic UI komponensek
- Navigáció, Elrendezések, Listák
- Ûrlap elemek Ionic módra
- Sokféle modális ablak

## API és SASS - elemek programozása és témázása
- Ionic komponensek kezelése
- Tulajdonásgok lekérése és módosítása
- Az SCSS preprocesszor és más állatfajták

## Ionic Native - a készülék elérése
- App és társai
- Geolocation és Contacts...

 Az alkalmazásunk mobil kliensének beüzemelése és egy kis build ismeret
